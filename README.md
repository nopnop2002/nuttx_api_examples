# nuttx_api_examples

nuttx api examples.   

#How to use   

git clone https://github.com/nopnop2002/nuttx_api_examples   
cd nuttx_api_examples   

nuttx directory structure.   
$HOME---NuttX-+-nuttx   
              +-apps---examples   

Copy all file to your NuttX/apps/examples directory.   

cd $HOME/NuttX/nuttx   
make clean   
make menuconfig
